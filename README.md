# To-Do List API (Node.js + Express + PostgreSQL)

## Features
- CRUD endpoints for tasks
- Cron job runs every 5 minutes to check for new tasks and send notifications
- Manual endpoint `/cron/trigger` to test cron behavior
- Postman collection included

## Requirements
- Node.js >= 18
- PostgreSQL
- (Optional) SMTP credentials for email notifications
